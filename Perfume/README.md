# Éclat — Luxury Perfume E-commerce

A mock perfume e-commerce site featuring luxury fragrances (Dior Sauvage, Armani, YSL, Chanel, Tom Ford, Creed, etc.) with a full shopping flow and mock checkout.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Features

- **Home** — Hero and featured fragrances
- **Shop** — Full catalog with Men / Women / Unisex filters
- **Product detail** — Add to cart, quantity selector
- **Cart** — Update quantity, remove items, order summary
- **Checkout** — Two-step mock checkout (shipping → payment), thank-you page

Checkout is **mock only**; no real payment is processed.
